# 挂树提醒插件 

> ## 功能介绍  
> 因 公主连结 游戏机制导致挂树时间限制1小时  
> 本插件通过粗略时间计算提醒玩家及时下树 避免掉刀  
> PS：插件轮巡机制支持多群同时使用  


>## 使用方法  
>开箱即用，插件放到modules下面，__bot__.py添加模块ontree_scheduler
>发送 *挂树 指令进行记录   
>若 挂树期限结束前下树 发送 *取消挂树 指令删除记录  
>若 挂树期限结束10分钟内仍未下树 BOT会三次*提醒下树 时间超过期限后记录自动删除
